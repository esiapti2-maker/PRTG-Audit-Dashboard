Consultas API & Comandos Útiles

Líneas de comando rápidas para interactuar con la API REST de PRTG de forma manual.

Consultar todos los Sensores (cURL):

curl "https://tu-prtg.gap.net/api/table.json?\ content=sensors\ &columns=objid,name,device,group,status,lastvalue,message\ &count=5000\ &username=e.siapti2\ &passhash=TU_PASSHASH"

Ejecución del Servidor Integrado (Python):

# Para iniciar la herramienta local e iniciar el Dashboard: python server.py # El servidor creará un historial CSV acumulado en: # /reportes_historicos/Auditoria_Sensores_*.csv
