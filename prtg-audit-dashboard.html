Scripts API — Consultas útiles

cURL — todos los sensores:

curl "https://tuprtg/api/table.json?\ content=sensors\ &columns=objid,name,device,group,status,lastvalue,message\ &count=5000\ &username=auditor\ &passhash=TU_PASSHASH"

cURL — dispositivos:

curl "https://tuprtg/api/table.json?\ content=devices\ &columns=objid,name,host,group,status\ &count=5000\ &username=auditor\ &passhash=TU_PASSHASH"

Python — auditoría multi-sitio:

import requests SITES = [ {"name":"GDL","host":"https://prtg-gdl.empresa.com","passhash":"HASH1"}, {"name":"MTY","host":"https://prtg-mty.empresa.com","passhash":"HASH2"}, # ... hasta 14 sitios ] for site in SITES: r = requests.get( site["host"]+"/api/table.json", params={"content":"sensors","count":5000, "username":"auditor", "passhash":site["passhash"]}, verify=False, timeout=30) data = r.json() down = [s for s in data["sensors"] if s["status"] != "Up"] print(f"{site['name']}: {len(down)} sensores down")
